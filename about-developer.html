Origin

The coding journey

From disliking programming to building decentralized software — a transformation in a little over two years.

Oct 2023

The turning point

Until my 9th-grade half-yearly exams I genuinely disliked coding — it seemed pointless and boring. Then one idea changed everything: what if I could make a character move across a terminal screen from user input?

Nov 2023

First breakthrough

My first real program — a character spawner that grew into an 8-line player character, animated by redrawing the terminal every 500ms so it could walk across the screen. All within a month.

Late 2023

First GUI applications

The terminal wasn't enough. I built my first GUI app — a Click-Per-Second tester — then my first game, Rex Run, inspired by Chrome's dinosaur game and Geometry Dash.

2024

The creative explosion

An unstoppable streak of 16 more applications: Ping Pong, Flying Dual, Flappy Bird, Memo, Tic Tac Toe, SG Paint, Notes, four desktop widgets, a JRE installer, a floating keyboard, and a color picker.

Jan 2024

The Talken vision

The idea for Talken was born — a privacy-first chat application to redefine how we think about digital communication. Months went into exploring the right technologies.

May 2025

First test build

The first testable build of Talken came together, laying the foundation for a privacy-first messaging experience.

Jun–Oct 2025

Decentralized architecture

Shipped the user-owned storage layer, lightweight coordination services, and a distributed directory with offline-first behavior — the backbone of Talken's serverless design.

Oct 20, 2025

Talken MVP launched Launched

Public release of the Talken MVP — a complete desktop application with 1:1 messaging, your-own-cloud storage, a multi-layer encryption design, and self-sovereign accounts.
